## bftpd

http://bftpd.sourceforge.net/


run without config file (use default settings) `bftpd -n -D`, default port is 21

config file
```
global{
	PORT="8855"
	LOGFILE="/tmp/1.log"
	ROOTDIR="/tmp/"
}
```
